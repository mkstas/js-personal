# Personal

## Описание

Сайт для представления пет проектов\
Проект размещен по адресу: https://vanilla-personal.vercel.app/

### Установка проекта

```sh
npm install
```

### Сборка для разработки

```sh
npm run dev
```

### Сборка готового проекта

```sh
npm run build
```

### Предпросмотр готового проекта

```sh
npm run preview
```
