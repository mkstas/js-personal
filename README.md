# Personal

## Описание

Сайт для представления личных работ веб-разработчика

### Установка проекта

```sh
npm install
```

### Сборка для разработки

```sh
npm run dev
```

### Сборка и минификация для продакшина

```sh
npm run build
```

### Предпросмотр собранного проекта

```sh
npm run preview
```
