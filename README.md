# Верстка Personal

## Описание

Сайт для размещения пет проектов\
Проект размещен по адресу: https://js-personal.vercel.app/

### Структура

- Приветствие
- Стек технологий
- Список пет проектов

## Установка

### Инсталляция

```sh
npm install
```

### Режим разработки

```sh
npm run dev
```

### Сборка приложения

```sh
npm run build
```

### Предпросмотр собранного проекта

```sh
npm run preview
```
